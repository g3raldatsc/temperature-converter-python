# Temperature Converter

A Python program that converts temperatures between Celsius, Fahrenheit, and Kelvin.  
The program provides an interactive menu that allows the user to select the desired conversion type.

---

## Description
This program helps users convert temperature values between the three most common scales:

- **Celsius (°C)**
- **Fahrenheit (°F)**
- **Kelvin (K)**

Users can choose from various conversion options through an interactive menu.  
Each result is displayed with two decimal places for clarity and precision.

Available conversions:
1. Celsius → Fahrenheit  
2. Celsius → Kelvin  
3. Fahrenheit → Celsius  
4. Fahrenheit → Kelvin  
5. Kelvin → Celsius  
6. Kelvin → Fahrenheit

## How to Run
1. Clone or download this repository.
2. Open your terminal in the project directory.
3. Run the program using:

```bash
python temperature_converter.py
```
